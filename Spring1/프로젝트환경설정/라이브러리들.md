# 스프링 부트 라이브러리들

라이브러리들이 의존관계가있네
뭘 설치했는데 다른라이브러리들도 달려서 다운되는. 그러한 의존관계가 있음

## starter web

- tomcat 웹서버 열어주는거.
- spring webmvc

## starter thymeleaf

- 템플릿엔진, html쉽게만들어주는 라이브러뤼

## starter(공통)

- 공통으로 가지고있는녀석들임
- spring-core
- logback, slf4j 로그관련 라이브러리들이구만
- 백에서는 console.log()같이 디버깅안하고 로그를 이용한데.

## 테스트라이브러리

- junit 이녀석이 테스트라이브러리중엔 젤중요
- mocito
- assertj
- spring-test
